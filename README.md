# Execute in Development

1. Clone this repository
2. Execute
```bash
npm install
```
3. Get NestJS CLI in your computer
```bash
npm i -g @nestjs/cli
```
4. Get up the database
```bash
docker-compose up -d
```

## Stack
* TypeScript
* NestJS
* MongoDB
